## Overview

This project develops a physics–machine learning framework for modeling and controlling geothermal borefield systems. The approach integrates physics-based thermal response models with neural networks to improve prediction accuracy while maintaining physical interpretability. The framework supports scalable modeling and enables advanced control strategies to improve the efficiency and reliability of geothermal energy systems.

**Key Methods:** `Scientific Machine Learning` · `Borefield Modeling` · `Advanced Control`

## Contributions
As a graduate research leader,  I:
- Developed a physics–machine learning framework for geothermal system modeling
- Implemented hybrid models combining g-function thermal response with neural networks
- Built predictive models for borefield outlet temperature and system dynamics
- On-going and future work: Applying the framework to optimization and advanced control of geothermal systems
